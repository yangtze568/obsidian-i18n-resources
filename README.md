# 🌐 社区插件 & 主题 AI 汉化指南

欢迎使用 AI 汉化服务！为了让您获得更好的使用体验，请在下载和使用前阅读以下说明。

## 🧩 插件汉化须知

- **⚠️ 确认原生中文：** 由于是 AI 汉化，部分已经自带官方中文的插件可能会出现“中翻中”的冗余情况。**请在下载前优先确认该插件是否已原生支持中文。**
    
- **🚧 汉化完整度：** 目前有部分插件的汉化工作仍在进行中，可能会遇到翻译不全的情况。
    

## 📦 译文版本与下载推荐

1.x 版本的插件目前已经整体迁移，在下载时您会看到不同的版本号，请参考以下说明：

| **译文版本号**   | **翻译方式**  | **质量说明**     |
| ----------- | --------- | ------------ |
| **`1.0.0`** | AI 自动提取翻译 | 未经过人工校对，翻译不全 |
| **`2.0.0`** | **人工校对**  | 经过人工校对，翻译更全  |

- 💡 **强烈推荐：优先下载 `2.0.0` 版本的译文！**
    
    即使 `2.0.0` 译文适用的插件版本略微落后于该插件的最新版本，人工校对的质量也能带来更顺畅的体验。
    

## 🔍 找不到特定插件的译文怎么办？

一些 1.x 版本的译文可能会出现通过“插件名称”搜索不到的情况。这通常是因为它们属于社区外的第三方插件，系统无法直接获取名称。

- **🛠️ 解决方案：** 尝试使用 **插件 ID** 进行搜索。
    
- **ℹ️ 如何获取插件 ID：** 插件 ID 即为该插件在本地 `plugin` 文件夹中所对应的**文件夹名称**。如brat插件所在地文件夹名称为obsidian42-brat
    

## 🎨 主题翻译与问题反馈

- **🤖 关于主题翻译质量：** 主题的 AI 翻译在历史版本中测试了几个不同的 AI 模型，因此翻译质量可能存在一定的波动（有好有坏）。
    
- **💬 期待您的反馈：** 您的反馈是我们改进的动力！如果您在使用过程中遇到任何离谱的翻译、排版错位或未能翻译的文本，欢迎随时提交反馈。
