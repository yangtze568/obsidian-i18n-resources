# 🌐 社区插件 & 主题 AI 汉化指南

欢迎下载obsidian i18n的翻译文件！为了让您获得更加顺畅的使用体验，请在下载和安装前花一分钟阅读以下指南。

## 🧩 插件汉化须知

- ⚠️ **注意避免“中翻中”**：由于本服务采用 AI 自动化处理，部分原生已经自带官方中文的插件可能会被系统重复翻译，产生冗余。建议您在下载汉化包前，优先确认该插件是否已原生支持中文。
    
- 🚧 **汉化完整度**：部分插件的汉化工作仍在不断迭代与完善中，您在使用时可能会偶尔遇到少量文本漏翻的情况，敬请谅解。
    

## 📦 译文版本与下载推荐（必看）

目前 1.x 版本的插件已整体迁移。在下载页面，您会看到不同的译文版本号。请参考下表了解版本差异：

|**译文版本号**|**翻译方式**|**质量说明**|
|---|---|---|
|`1.0.0`|🤖 AI 自动提取翻译|未经人工校对，可能存在机翻感或翻译不全的问题|
|`2.0.0`|👤 人工校对|**经过人工精校，翻译更全面、表达更自然**|

### 💡 如何选择最合适的翻译版本？

在面对多个下载选项时，请遵循**“先看译文版本，再看插件版本”**的原则。



以 **Latex Suite** 插件为例（如配图所示），假设您看到了以下三个选项：

1. 译文 `v1.0.0` [对应插件 `1.11.0`]
    
2. 译文 `v2.0.0` [对应插件 `1.9.8`] 👉 **（推荐选择项）**
    
3. 译文 `v2.0.0` [对应插件 `1.9.5`]


**✅ 选择逻辑如下：**

1. **优先认准 `2.0.0`**：虽然第一个选项的插件版本（1.11.0）是最新的，但它的译文质量只有 `1.0.0`。人工校对的 `2.0.0` 质量远优于纯 AI 提取，因此我们直接排除第一个选项，在后两个 `2.0.0` 中进行选择。
    
2. **其次选择最新的插件版本**：对比后两个选项，`[1.9.8]` 比 `[1.9.5]` 的版本更新。
    
3. **最终结论**：综合考虑，**强烈建议选择第二个翻译文件**。即使它的插件版本略微落后于最新的 1.11.0，但高质量的人工精校绝对能为您带来更舒适的使用体验！
    

## 🔍 找不到特定插件的译文怎么办？

部分早期版本的译文（或是未上架官方社区的第三方插件）可能会出现通过“插件名称”搜索不到的情况，这是因为系统无法直接匹配到名称。

- 🛠️ **解决方案**：请尝试使用**插件 ID** 进行精准搜索。
    
- ℹ️ **如何获取插件 ID？** 插件 ID 就是该插件在您本地 `plugins` 文件夹中所对应的文件夹名称。例如：BRAT 插件所在的文件夹名称为 `obsidian42-brat`，那么 `obsidian42-brat` 就是它的插件 ID。
    

## 🎨 主题翻译与问题反馈

- 🤖 **主题翻译质量说明**：在历史版本的主题 AI 翻译工作中，我们测试了多种不同的 AI 模型。因此，不同主题之间的翻译质量可能会存在一定的波动（部分可能不够自然）。
    
- 💬 **期待您的反馈**：您的反馈是我们不断优化的最大动力！如果在日常使用中遇到任何离谱的翻译、界面排版错位，或是未被翻译的英文漏网之鱼，欢迎随时向我们提交反馈。
